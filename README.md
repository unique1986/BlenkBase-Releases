# BlenkBase Releases

Öffentliche Release- und Updatequelle für **BlenkBase – Command Your World**.

- `update-manifest.json` enthält verfügbare stabile und Beta-Versionen.
- Die Windows-Pakete liegen als Assets in den zugehörigen GitHub Releases.
- BlenkBase prüft HTTPS-Quelle, Repository-Zuordnung, Paketgröße, ZIP-Pfade und SHA-256.
- Schlägt ein Programmupdate fehl, stellt der lokale Updatehelfer automatisch die vorherige Version wieder her.

Das eigentliche BlenkBase-Projekt wird zunächst privat entwickelt. Dieses Repository enthält ausschließlich die für Online-Updates benötigten öffentlichen Releaseinformationen.
